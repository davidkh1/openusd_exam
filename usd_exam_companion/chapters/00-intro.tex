\section{Role Summary}
An OpenUSD developer builds, maintains, and optimises 3D pipelines using OpenUSD across industries such as animation, VFX, gaming, manufacturing, AECO, and virtual production.

\section{Key Responsibilities}
\begin{itemize}[leftmargin=*]
  \item Develop and maintain OpenUSD-based pipelines and tooling.
  \item Integrate OpenUSD with DCC tools; optimise asset performance.
  \item Automate tasks; troubleshoot pipeline issues; document workflows.
  \item Track and adopt OpenUSD advancements.
\end{itemize}

\section{Recommended Qualifications}
\begin{itemize}[leftmargin=*]
  \item Strong OpenUSD knowledge; Python/C++ programming.
  \item 3D fundamentals; version control; debugging mindset.
  \item Rendering/material familiarity; clear communication across teams.
\end{itemize}
