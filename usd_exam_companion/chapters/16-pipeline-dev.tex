\CheatSheetTitle
\begin{itemize}[leftmargin=*]
  \item \textbf{Asset structure}: stable root prim, predictable paths, consistent kind/model hierarchy.
  \item \textbf{Collaboration}: separate “authored source” from “published” layers; use sublayers for departments.
  \item \textbf{Validation gates}: enforce rules at publish time, not when scenes break downstream.
\end{itemize}
